PyMergence documentation
========================

PyMergence is a Python package for calculating and visualizing causal emergence in complex systems.

.. note::
   This package is under active development---API changes may occur, and more features will be added. If you have any questions or suggestions, please open an issue on the GitHub repository.

Installation
------------

Install PyMergence from source:

.. code-block:: bash

   git clone https://github.com/yourusername/pymergence.git
   cd pymergence
   pip install -e .

Pip install is not yet available, but will be in the future.

Quick Start
-----------

A quick example to get you started is available as an interactive Jupyter notebook in the `Tutorial.ipynb` file at `https://github.com/AJnsm/pymergence/blob/main/Tutorial.ipynb <https://github.com/AJnsm/pymergence/blob/main/Tutorial.ipynb>`_.
